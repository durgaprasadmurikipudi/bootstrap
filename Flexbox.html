To start using flex box in bootstrap we have to mention the class d-flex on the container that you want to apply

justify-content in flexbox affects the horizontatl alignment of the children in the container

align-items affects the vertical alignment of the children in the container

Direct children means, the direct siblings under the container, not the children inside the siblings.

We are gonna create a row and col inside it. Inside col there will be three paragraphs, notice the alignment of those paragraphs with different justify content variations of flexbox.

The col is given d-flex class and justify-content-start config, notice on use of d-flex the p tags inside, became inline alignments and aligned side by side.

The col is given d-flex class and justify-content-start config, but notice the elongation of elements, this is the default behaviour of elements if we did not mention align-items* config, the flex items elongate their height as accordingly to the height of the container. The container for demonstration purpose was set to have 150px height. Notice that we need not hard code height 150px as perse, we can also set min-height and it will occupy as much height it can not going below min-height. use vh units isntead of px for more consistency across screens.
