Introduction to basic grid system
Grid system in boostrap is divided in 12 equidistant columns. A col class mention on a div with out any size will simple occupy all 12 columns width. Like this one.

But if you mention 2 columns without any mention of size they each occupy 6 column units.
Thus way paving way to basic grid system.

I can even mention that this division is only valid only above certain screen sizes
For example this column division wont be visible if you reduce the screen size to less than 576px as this uses col-sm class.
